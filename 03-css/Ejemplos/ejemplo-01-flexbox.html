Guía Completa de Flexbox

¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño unidimensional que permite distribuir espacio y alinear elementos en una interfaz. Es especialmente útil para componentes de aplicaciones y estructuras de pequeña escala.

La idea principal detrás de Flexbox es dar al contenedor la habilidad de alterar la anchura/altura y el orden de sus elementos para ocupar mejor el espacio disponible.

1. Contenedor Flex Básico

Para empezar a usar Flexbox, definimos un elemento como contenedor flex:

.flex-container {
    display: flex;
}

5. Propiedades de Crecimiento y Reducción

Las propiedades flex-grow, flex-shrink y flex-basis controlan cómo los elementos crecen y se reducen:

flex-grow:

Determina cuánto puede crecer un elemento si es necesario:

flex-grow: 1; /* El elemento crecerá */
flex-grow: 2; /* Crecerá el doble que otros con flex-grow: 1 */
Normal
flex-grow: 1
flex-grow: 2

flex-shrink:

Determina cuánto puede reducirse un elemento si es necesario:

flex-shrink: 1; /* Valor por defecto, el elemento puede reducirse */
flex-shrink: 0; /* El elemento no se reducirá */
flex-shrink: 0 (No se reduce)

Conclusión

Flexbox es una herramienta poderosa para crear diseños flexibles y responsivos. Es especialmente útil para:

  • Alineación vertical y horizontal de elementos
  • Distribución de espacio entre elementos
  • Ordenamiento de elementos independientemente del orden en el HTML
  • Diseños que deben adaptarse a diferentes tamaños de pantalla

Para diseños más complejos en dos dimensiones (filas y columnas simultáneamente), considera usar CSS Grid.
