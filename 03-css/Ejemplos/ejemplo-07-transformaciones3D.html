Transformaciones y Efectos 3D en CSS

Transformaciones 2D

Las transformaciones 2D permiten modificar elementos en el plano bidimensional. Pasa el cursor sobre cada caja para ver el efecto.

Scale

transform: scale(1.2);

Rotate

transform: rotate(45deg);

Skew

transform: skew(10deg, 10deg);

Translate

transform: translate(20px, -20px);

Múltiple

transform: scale(1.1) rotate(10deg) translateX(10px);

Origin

transform-origin: top left; transform: rotate(30deg);

Transformaciones 3D Básicas

Las transformaciones 3D permiten manipular elementos en el espacio tridimensional. Pasa el cursor sobre cada caja para ver el efecto.

RotateX

transform: rotateX(60deg);

RotateY

transform: rotateY(60deg);

RotateZ

transform: rotateZ(60deg);

Translate3D

transform: translate3d(20px, -20px, 50px);

Un cubo 3D completo creado con transformaciones CSS 3D. Pasa el cursor sobre el cubo para verlo girar.

• perspective: 1000px
