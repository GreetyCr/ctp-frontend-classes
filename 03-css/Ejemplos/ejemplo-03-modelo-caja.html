Modelo de Caja en CSS

1. Modelo de Caja Estándar vs. Border-Box

El modelo de caja estándar incluye el ancho/alto + padding + borde (el margen es externo).

Modelo Estándar (content-box)

Este elemento tiene un ancho total de 360px (300px + 20px*2 + 10px*2) 
width: 300px;
height: 150px;
padding: 20px;
border: 10px solid #3498db;
margin: 30px;
/* box-sizing: content-box; (por defecto) */

Modelo Alternativo (border-box)

Este elemento tiene un ancho total de 300px (ancho incluye padding y borde) 
box-sizing: border-box;
width: 300px;
height: 150px;
padding: 20px;
border: 10px solid #9b59b6;
margin: 30px;

4. Colapso de Márgenes (Margin Collapse)

Cuando dos elementos verticales se encuentran, sus márgenes colapsan, tomando el valor del margen mayor.

Margen inferior: 30px
Margen superior: 40px

El espacio total entre los dos elementos es 40px (no 70px)

/* Elemento 1 */
margin-bottom: 30px;

/* Elemento 2 */
margin-top: 40px;

5. Centrado Horizontal con Margin Auto

Elemento centrado
width: 300px; /* Debe tener un ancho definido */
margin: 0 auto; /* Centra horizontalmente */

7. Outline (contorno)

A diferencia del borde, el outline no afecta al layout y puede tener un offset.

Ejemplo de outline
border: 2px solid #3498db;
outline: 4px dashed #e74c3c;
outline-offset: 5px;

11. Propiedades de Visibilidad

Visible
Hidden
Collapse

A diferencia de display:none, visibility:hidden mantiene el espacio del elemento en el layout.

visibility: visible; /* Valor por defecto */
visibility: hidden; /* Invisible pero ocupa espacio */
visibility: collapse; /* Similar a hidden, con excepciones */

Elementos Inline
