2. Unidades fr (fracciones)

Usando unidades fr para dividir el espacio disponible:

1fr
2fr
1fr
grid-template-columns: 1fr 2fr 1fr;

3. Función repeat()

Usando repeat() para crear patrones repetidos:

1
2
3
4
5
6
grid-template-columns: repeat(3, 1fr);

4. Filas Explícitas

Definiendo explícitamente filas con diferentes alturas:

1
2
3
4
5
6

grid-template-columns: repeat(3, 1fr);

5. Filas Automáticas

Usando grid-auto-rows con minmax() para filas de altura variable:

Contenido corto
Contenido que podría ser más largo y necesitar más espacio
Contenido
Otro elemento con contenido variable que podría necesitar más espacio vertical
Contenido corto
Contenido
grid-template-columns: repeat(3, 1fr);
grid-auto-rows: minmax(80px, auto);

6. Espaciado con Gap

Usando gap para separar filas y columnas uniformemente:

2

4
5

7. Gap Específico

Configurando diferentes espaciados para filas y columnas:

1
2
3
4
5
6

8. Grid Template Areas

Creando layouts con nombres de área:

Header
Sidebar
Main Content
Footer
grid-template-columns: repeat(4, 1fr);
grid-template-rows: 80px 200px 80px;
grid-template-areas:
    "header header header header"
    "sidebar main main main"
    "footer footer footer footer";

9. Posicionamiento de Elementos

Posicionando elementos individualmente en la cuadrícula:

1 (columnas 1-2, fila 1)
2 (columna 3, filas 1-2)
3 (columna 1, fila 2)
4 (columna 2, fila 2)
5 (columnas 1-3, fila 3)
/* Ejemplo para el item 1 */
grid-column: 1 / 3; /* Desde línea 1 hasta línea 3 */
grid-row: 1;

12. Auto-fit vs Auto-fill

auto-fit (ajusta elementos para llenar el espacio):

1
2
3
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

auto-fill (mantiene espacio para elementos adicionales):

1
2
3
grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
